Basics of the web

The web contains: the web, HTML, URLs, HTTP, and Web apps. Major pieces are your computer + browser (IE, Firefox), internet cloud, and servers. The internet uses a protocol of HTTP to send/receive messages. Your computer send a HTTP request to web server then server return the document coded in HTMLwhich your web browser then displays to your screen. This video does a good job of explaining.

HTML

Hypertext Markup Language is the most common coding used on the web. It has two parts to it, TEXT which is actually the written words you see and MARKUP which gives style to the contents. There can be references to images or other pages.

Principle parts of HTML

There are tags which most require an opening tag and a closing tag with content between them, such as Bold or Emphasis. As you can tell the tags can define how the content looks.The tags along with the content are called an element. There are several different tags such as Anchor which reference a hyper-link. The Image tag will allow you to define a URL however it doesn't require a closing tag. The Break tag will allow you to add white space to the screen.

Types of Elements

There are two types of elements inline and block. The block elements have an invisible box around the contents and seen here by the paragraphs.
